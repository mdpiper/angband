## usage notes

Installed through Homebrew.
The user directory is

    ~/.angband/Angband

It doesn't render correctly in iTerm2.
Use an xterm instead:

    xterm -fa Monaco:size=18 -rv &

See available fonts:

    fc-list | cut -f2 -d: | sort -u

Or use Terminal.
